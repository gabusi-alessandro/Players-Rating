# Players-Rating
L'obiettivo del progetto è quello di costruire modelli statistici per la valutazione delle prestazioni dei giocatori a partire dalle statistiche estratte da sofascore.

Il progetto si compone di 5 fasi principali:
- Scraping dei dati da https://www.sofascore.com/
- Creazione del database a partire dai dati estratti
- Costruzione dei modelli
- Scraping dati delle nuove partite e caricamento di questi nuovi dati nel database
- Valutazione della prestazione dei giocatori

Nelle prossime settimane pubblicherò su subsatck una serie di post per dare una spiegazione più precisa e dettagliata di ogni fase del progetto. (https://substack.com/@alessandro344866)
